# 知识图谱实体与关系类型说明

本文档根据当前 `data/raw/ai_knowledge.json` 统计整理，用于说明本项目知识图谱中定义的实体类别和关系类型。

## 一、实体类别（Entity）

当前图谱共定义 12 类实体：

| 英文类型 | 中文说明 | 当前数量 |
|---|---|---:|
| `Technology` | 技术方向 | 160 |
| `Model` | 大模型 / 模型 | 61 |
| `Application` | 应用场景 | 39 |
| `Organization` | 研发机构 / 企业 | 33 |
| `Product` | AI 产品 | 23 |
| `Algorithm` | 核心算法 / 架构 | 19 |
| `Hardware` | 硬件 / 芯片 | 14 |
| `Policy` | 政策 / 规范 | 12 |
| `Person` | 关键人物 | 6 |
| `Infrastructure` | 基础设施 | 6 |
| `Metric` | 评测指标 / 优化指标 | 3 |
| `Problem` | 技术问题 | 1 |

可按答辩口径概括为：

```text
实体类别包括：模型、机构、技术、算法、指标、问题、产品、应用、硬件、基础设施、政策和人物。
```

## 二、关系类型（Relation）

当前图谱中实际使用的关系类型包括：

| 中文关系 | 英文映射 | 含义说明 |
|---|---|---|
| `研发` | `DEVELOPS` | 某机构、企业或团队研发某模型、产品或技术 |
| `使用架构` | `USES_ARCH` / `USES_TECH` | 某模型采用某种核心架构或技术机制 |
| `优化` | `OPTIMIZES` | 某技术对某目标、能力或过程进行优化 |
| `优化指标` | `OPTIMIZES_METRIC` | 某指标用于衡量或优化某类问题 |
| `提升` | `IMPROVES` | 某技术提升某种能力或性能 |
| `降低` | `REDUCES` | 某技术降低成本、延迟、显存或风险 |
| `缓解` | `MITIGATES` | 某技术路径缓解某类问题，如模型幻觉 |
| `支持` | `SUPPORTS` | 某技术、模型或基础设施支持某能力 |
| `支撑` | `SUPPORTS` | 某基础能力或设施支撑某模型、技术或应用 |
| `基于` | `BASED_ON` | 某实体建立在另一实体基础之上 |
| `属于` | `BELONGS_TO` | 某实体属于某类别或技术体系 |
| `应用` | `APPLIES_TO` | 某技术应用于某场景，或某应用使用某技术 |
| `实现` | `IMPLEMENTS` | 某模型或产品实现某类能力 |
| `规范` | `REGULATES` | 某政策或标准规范某类技术或应用 |
| `研究` | `RESEARCHES` | 某人物、机构或实体研究某方向 |
| `发布` | `RELEASES` | 某机构发布某模型、产品或标准 |
| `依赖` | `DEPENDS_ON` | 某实体依赖另一实体提供能力 |
| `提出` | `PROPOSES` | 某人物或机构提出某理论、方法或技术 |
| `改进` | `IMPROVES` | 某方法对已有技术进行改进 |
| `扩展` | `EXTENDS` | 某技术在已有能力基础上扩展 |
| `增强` | `ENHANCES` | 某技术增强某种能力 |
| `推动` | `PROMOTES` | 某机构、政策或技术推动某领域发展 |
| `孵化` | `INCUBATES` | 某机构孵化某企业或项目 |
| `提供` | `PROVIDES` | 某机构或平台提供某能力、服务或资源 |
| `建设` | `BUILDS` | 某主体建设某基础设施或平台 |

可按答辩口径概括为：

```text
关系类型包括：研发、采用技术、优化目标、缓解问题、支撑能力、应用场景、基于、属于、发布、规范和推动等。
```

## 四、本体设计说明

本项目的知识图谱不是单纯的“概念词典”，而是围绕人工智能技术链条组织实体与关系：

```text
机构 / 企业 → 研发 → 模型 / 产品
模型 → 使用架构 → 核心算法 / 技术机制
技术机制 → 优化 / 提升 / 降低 → 性能目标
技术路径 → 缓解 → 技术问题
技术 / 产品 → 应用 → 应用场景
政策 / 标准 → 规范 → 技术或应用
```

这种设计能够支持以下类型的问题：

```text
DeepSeek-R1 如何增强逻辑推理能力？
Qwen3 使用了哪些架构？
Kimi K2 和多头潜在注意力有什么关系？
解决大语言模型幻觉的技术路径有哪些？
FlashAttention 主要优化了什么？
推测解码为什么能提升推理加速？
```
